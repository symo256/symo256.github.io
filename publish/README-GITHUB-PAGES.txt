How to publish this website with GitHub Pages

1. Go to https://github.com and sign in.
2. Create a new public repository. A good name is simonz-tours-and-trevel.
3. Open the new repository and choose Add file, then Upload files.
4. Upload these website files:
   - index.html
   - about.html
   - services.html
   - contact.html
   - styles.css
   - script.js
   - .nojekyll
5. Click Commit changes.
6. Go to Settings, then Pages.
7. Under Build and deployment, choose Deploy from a branch.
8. Select the main branch and the root folder, then save.
9. Wait a minute or two. GitHub will show your public website link on the Pages screen.

Your website link will look like:
https://your-github-username.github.io/simonz-tours-and-trevel/
